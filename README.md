# Tic-Tac-Toe with Minimax

A Tic-Tac-Toe game built with Pygame featuring an unbeatable Minimax algorithm.

## Features

- Play against an algorithm
- Choose to play as X or O
- Visual game board with Pygame interface

## Requirements

- Python 3.x
- Pygame

## Setup

1. Clone the repository:
```bash
git clone https://github.com/Elie-Soued/tictactoe.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the game:
```bash
python3 ./runner.py
```

## How to Play

1. When the game starts, choose whether to play as X or O
2. Click on an empty cell to make your move
3. The machine will automatically make its move
4. The game ends when someone wins or the board is full
5. Click "Play Again" to start a new game
